# gol
gol is simple application lancher.

## Support OS
- Mac

## Installtion
```
$ git clone https://github.com/skanehira/gol
$ cd gol
$ go build
```

## Usage
```
# default
$ gol

# fzf mode
$ gol -f

# output application list
$ gol -o

# run specified application
$ gol -s safari

# run specified applicatiions
$ gol -s safar,notes
```
